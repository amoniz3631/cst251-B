Lab 19: CSS Frameworks

Challenge

The challenges brought on by this lab were more fun than frustrating. Usually while doing labs we are challenged by things not working properly, while in this lab things either worked right away or didn't work at all making any challenges that came up much more researchable. The dropdowns and accordion menus were a bit challenging at first but ended up being simpler to quickly implement.

Problems

We didn't run into any significant problems, anything we wanted to do felt possible using mostly the Bootstrap website. For example I wanted to have multiple submenus within my "previous labs" dropdown, I quickly found a tutorial on bootstrap-menu which showed exactly how to do it. Usually it takes much more collaboration and Googling to find solutions that are often too complex to realistically implement.

Results

This lab taught us how to use the basic framework of Bootstrap to build a simple webpage including a navbar with dropdowns. We can now use the different classes to specify our intentions for things such as centering, margins, colors, and more. Finding solutions was like solving a puzzle and makes coding less frustrating and more exciting.
